# Jquery(Javascript)
1. 非法字符(Illegal character)与网址链接相互转换的JS方法
	+ encodeURI("非法字符")
	+ decodeURI("网址链接")
	+ 例子
		+ encodeURI("哈喽")=%E5%93%88%E5%96%BD
		+ encodeURI("%E5%93%88%E5%96%BD")=哈喽
2. Jquery添加与移除html元素
	+ $("#elementId").append("html content")
	+ $("#elementId").html("")
	+ 例子
		+ $("#selecId").append("<option value=\" + value + "\">value</option>")
		+ $("#selecId").html("")
